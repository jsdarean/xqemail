🔒 本地方案：邮件数据不经过任何第三方，直接通过公司 SMTP 发送。 使用前需启动本地中继服务。

⚙️ SMTP 邮箱配置

配置你的企业邮箱 SMTP 信息，授权码将保存在本地 Chrome 存储中。

📋 默认抄送邮箱

设置默认抄送邮箱，每次打开发送页面时自动填入。

✍️ 邮件签名

编辑你的邮件签名，发送邮件时会自动追加到正文末尾。支持 HTML 格式。

📁 IMAP 已发送存档

发送邮件后，通过 IMAP 将副本存档到邮箱的「已发送」文件夹，这样网页邮箱也能看到发送记录。

ℹ️ IMAP 账号密码与 SMTP 相同，无需单独填写。服务器地址留空时自动从 SMTP 地址推导（smtp.xxx → imap.xxx）。

📖 本地服务器说明
✅ 已设置为开机自动启动（静默后台运行，无窗口），无需每次手动操作。
  1. 服务器已加入 Windows 启动项，开机后自动在后台运行
  2. 如需手动启动：进入 local-smtp-server 目录双击 start-server.bat
  3. 打开本插件后，在邮箱/数据库设置中查看服务器状态
🔒 安全保障：
• 服务器仅监听 127.0.0.1（本机），外部网络无法访问
• 邮件内容直接从你的电脑发往公司 SMTP 服务器，不经任何第三方
• 授权码仅存储在本地 Chrome 存储中
🗄️ MySQL 数据库设置

配置本地 MySQL 数据库连接，发送邮件后会自动将需求信息写入数据库。
